## 📌 Description
The Placement Portal is a web-based application designed to manage student placements efficiently. It allows students to explore job opportunities and enables administrators to manage placement-related data in a structured way.

---

## 🚀 Features
- 🔐 User Authentication (Login/Register)
- 👨‍🎓 Student Profile Management
- 🏢 Company / Job Listings
- 📅 Placement & Interview Management
- 📊 Dashboard for tracking placement data
- 🔎 Easy navigation and user-friendly interface
